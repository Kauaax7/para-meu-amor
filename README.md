# 🎵 Nossa História - Presente de Aniversário de Namoro

# Um site interativo, mobile-first, desenvolvido com a estética oficial do
# Spotify. Este projeto foi criado como um presente personalizado para celebrar o
# tempo juntos, relembrando qualidades e momentos especiais através de música,
# mensagens e fotos.
#
# 📸 Visualização
#
 # - Tema: Dark Mode (Spotify Black).
 # - Cores: Verde Spotify (#1DB954) e Rosa Vibrante (#E8115B).
 # - Tipografia: Montserrat (via Google Fonts). 
#
# ✨ Funcionalidades
#
#  - Contador em Tempo Real: Um cronômetro que calcula dias, horas, minutos e
#    segundos desde o início do namoro (05/01/2025).
#  - Slides Narrativos: Transições suaves entre telas com mensagens
#    personalizadas e versículos bíblicos.
#  - Player de Música: Integração com o Spotify Embed para tocar a trilha sonora
#    do casal diretamente no site.
#  - Galeria de Fotos Dinâmica: Sistema de upload de fotos com persistência via
#    localStorage.
#  - Painel Administrativo: Botão discreto protegido por senha para gerenciamento
#    de fotos.
#
# 🛠️ Tecnologias Utilizadas
#
#  - HTML5 Semântico
#  - CSS3 (Flexbox, Grid, Animações e Mobile-first)
#  - JavaScript Vanilla (sem frameworks)
#  - Spotify Embed API para o player de música.
#  - FileReader API para processamento de imagens localmente.
#
# 🚀 Como usar
#
# 1. Preparação
#
# Basta abrir o arquivo index.html em qualquer navegador moderno (Chrome, Safari,
# Edge).
#
# 2. Adicionando suas fotos (Área do Namorado)
#
# Para que o site tenha as fotos de vocês antes de você mostrar para ela, siga
# estes passos:
#
# 1.  No canto superior direito, existe um botão rosa "+" discreto.
# 2.  Clique nele e digite a senha padrão: amor2025.
# 3.  Selecione a foto no seu dispositivo.
# 4.  Insira uma legenda ou a data do momento.
# 5.  Atenção: As fotos ficam salvas no navegador onde foram carregadas
#    (armazenamento local).
#
# Dica importante: Como o site usa localStorage, se você carregar as fotos no seu
# computador, elas não aparecerão automaticamente no celular dela.
#
#  - Recomendação: Abra o link no celular dela (ou no dispositivo que for
#    entregar), use o botão "+" e suba as fotos diretamente lá uma vez. Elas
#    ficarão salvas para sempre naquele aparelho.
#
# 3. Navegação
#
#  - Utilize os botões "Anterior" e "Próximo" na parte inferior.
#  - No celular, você também pode deslizar para os lados (swipe) para trocar de
#    slide.
#
# 📖 Estrutura de Conteúdo
#
#  - Slide 1: Abertura com Player de Música.
#  - Slide 2: Timeline e Contador de tempo.
#  - Slide 3-6: Slides temáticos (Olhos, Sorriso, Cuidado, Futuro) com versículos
#   de Cantares, Provérbios, Coríntios e Jeremias.
# - Slide 7: Grid de fotos personalizado.
# - Slide 8: Mensagem final de Feliz Aniversário.
#
# ❤️ Criado com amor para um momento especial.
